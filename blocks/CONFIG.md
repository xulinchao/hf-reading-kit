# v2 Block 调参指南

每个 block 文件顶部 `<style>` 里都有 **`:root` CSS 变量**，改数字即可，不用动布局代码。

## 通用变量（多数 block 共有）

| 变量 | 含义 | 默认 |
|------|------|------|
| `--zone-top` | 组件距画面顶部 | `8%` |
| `--zone-h` | 组件可用高度 | `44%` |
| `--wrap-w` | 卡片总宽度 | 各组件不同 |

| **book-cover.html** | `--book-w` 封面宽度 | `340px` |

## chart-waves.html

| 变量 | 含义 | 默认 |
|------|------|------|
| `--wrap-w` | 时间轴卡片宽度 | `1140px` |
| `--title-size` | 标题字号 | `40px` |
| `--icon-size` | 图标方块边长 | `96px` |
| `--label-size` | 平台名字号 | `38px` |
| `--years-size` | 「13 年」字号 | `22px` |
| `--track-h` | 进度条高度 | `10px` |

## card-tags-oss.html

| 变量 | 含义 | 默认 |
|------|------|------|
| `--zone-top` | 组件距画面顶部 | `20%` |
| `--card-w` | 单卡宽度 | `780px` |
| `--name-size` | 主标题字号 | `50px` |
| `--stack-offset` | 底卡下移（露出叠放感） | `80px` |
| `--slap-from-y` | 拍入起始高度 | `-56px` |

拍入时间：`T1=1.533s` / `T2=3.233s`（相对 block 起点 87.7s，对齐旁白 89.2s / 90.9s）

## card-signal-*.html

| 变量 | 含义 | 默认 |
|------|------|------|
| `--card-w` | 信号卡宽度 | `820px` |
| `--title-size` | 主标题 | `42px` |

## 主画面 overlay（非 block）

在 `style.css` 的 `#stage` 里：

| 变量 | 含义 | 默认 |
|------|------|------|
| `--u` | 中间大字倍率 | `2.12` |
| `--sub` | 字幕字号（已禁用） | `22px` |

## 时间轴 beat 数据

`scenes-data.js` → `beats` 数组：改 `s` / `e` 对齐旁白。
